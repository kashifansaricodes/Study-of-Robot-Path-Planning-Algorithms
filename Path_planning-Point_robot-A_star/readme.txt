#Steps to run the program:

1. To run the a_star.py use any code editor of your choice, VSC, Spyder or Colab is preffered. 
2. Run the program and close the window of inital_map to proceed ahead.
3. Input stepsize
4. Input clearance (her it will apear blue in color)
5.Input robot radius (this will be added to clearance)
6. Input start co-ordinates.
7. Input goal co-ordinates.
8. Please wait for path to reach the goal and an output will be backtracked.
9. Finally the scanned map will appear with the generated path.
